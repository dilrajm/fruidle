# Fruidle

**Fruidle** is a browser-based fruit-themed puzzle game built with HTML, CSS, and JavaScript. The game challenges players to interact with fruit-based clues and gameplay logic through a simple, clean, and responsive web interface.

## Live Demo

Play the game here:
https://dilrajm.github.io/fruidle/#Homepage

## GitHub Repository

View the source code here:
https://github.com/dilrajm/fruidle

## Features

* Interactive browser-based gameplay
* Fruit-themed puzzle/game concept
* User input handling and game-state updates
* Dynamic visual feedback based on player actions
* Custom styling with CSS
* Organized image assets for the game interface
* Deployed publicly using GitHub Pages

## Technologies Used

* HTML
* CSS
* JavaScript
* Git
* GitHub
* GitHub Pages

## Project Purpose

With the help of my teamates, this project was created as a hands-on programming project to practice front-end development, JavaScript logic, user interaction, and browser-based game design. Fruidle helped strengthen my understanding of how HTML, CSS, and JavaScript work together to create a complete playable web application.

## What I Learned

While building Fruidle, I practiced:

* Structuring a front-end web project
* Writing JavaScript logic for interactive gameplay
* Handling user actions and updating the page dynamically
* Debugging game logic and interface issues
* Managing project files and assets
* Using Git and GitHub for version control
* Deploying a live web project with GitHub Pages
