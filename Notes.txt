Dilraj Mukkar

* Navigation & HTML Structure
- Coded homepage and level buttons to switch between sections.
- Ensured a single HTML file is used, showing only the active page and hiding others.
- Implemented help button pop-up with blurred background styling.
- Restructured HTML into sections (homepage, levels, modal controls) with proper indentation and comments.

* Core Game Functionality
- Coded all game functionality except timer, win/lose pop-ups, and scroll lock.
- Created a `levels` object mapping number of fruits and available images to each level.
- Implemented delete button functionality, updating game state and preventing deletion on empty cells.
- Implemented enter button to check row completion.
- Developed randomized secret key generator selecting fruits from level-specific pools with duplicates, ensuring correct array lengths.
- Input system: click-based fruit selection for next available grid cell, with out-of-bounds prevention.
- Row validation: enter allowed only when rows are full; compares guess vs secret key with Wordle-style logic.
- Two-pass evaluation: first pass marks correct placement (green), second pass marks present but misplaced (yellow), avoiding duplicate counting.
- Grid rendering: dynamically updates grid tiles and applies CSS visuals consistently.
- Button lock: disables buttons when game is won or lost.
- Button enabling: resets grid availability when loading a new level.
